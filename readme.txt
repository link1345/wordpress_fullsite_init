このテーマは、書籍「WordPressフルサイト編集」で付録として提供しているテーマです。

== Installation ==

1. WordPress上の管理者メニューの「外観」から「テーマ」の「新規作成」ボタンをクリック
2. 「テーマのアップロード」ボタンをクリック
2. このテーマのzipファイルを選択して、インストール
